# Guías Lu4 — Lineage 2 (MasterWork)

Guía interactiva de progresión **nivel 1 → 85** para el servidor Lu4.

Elige **raza → clase** y ves misiones, mejores zonas de farmeo, Path, Templo, marcas, Saga y reglas del servidor.

## Abrir la guía

**Web (GitHub Pages):**  
https://mathiasalejandr0.github.io/L2_GuiaPro_lu4/

**Local:** abre `index.html` (necesita `lu4-guides.js` y `lu4-chat.js` en la misma carpeta).

## Archivos

| Archivo | Qué es |
|---|---|
| `index.html` | Dashboard (raza → clase → guía + chat) |
| `lu4-guides.js` | Datos y guías por clase |
| `lu4-chat.js` | Asistente local |
| `GUIA_LU4_TODAS_LAS_CLASES.md` | Guía maestra en Markdown |
| `FICHAS_SPOILER_DESTROYER_STORM_SCREAMER.md` | Fichas detalladas Spoiler / Destroyer / Storm Screamer |

## Fuente

Todo basado en la wiki oficial Lu4: [masterwork.wiki/lu4](https://masterwork.wiki/lu4/main)
